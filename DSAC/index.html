Description



The objective of this work is to detect the contours of buildings using the model DSAC (Deeep structured active contour), the model is trained on Vaihingen buildings dataset, it consists of 168 buildings extracted from the training set of the ISPRS “2D semantic labeling contest”, 100 buildings are used to train the models and the remaining 68 as a test set.
The images shown above are the results of contour delineating with manual initialization, that means the user have to click on the center of the building in order to generate contour.
The initial contour is in blue, the result in yellow, and the ground truth in green. With the data term we can observe regions of lower energy along the boundary of the building. The balloon term has learned to produce positive values only inside the building, especially next to corners. The smoothness term penalize the curative that are far from the building’s corners. The elasticity term provided by the model in this example was α(x) = 0.74.
The value of Intersection over union in this test example is IoU = 0.63.



DSAC Architecture



The CNN predicts the values of the energy terms to be used by the active contour model (ACM): a global α for the length penalization and maps for local D, the data term, β, the curvature penalization and κ, the balloon term.
After ACM inference, a structured loss is computed and given to the CNN, whose parameters can then be updated using backpropagation.

Conclusion



Results show that model DSAC which combines extraction capabilities of CNNs with the high-level constraints provided by ACMs is a promising path for instance segmentation when we have geometric priors like buildings.
